用 root 帐户运行下面的命令，添加证书

wget -q -O - https://dl-ssl.google.com/linux/linux_signing_key.pub | apt-key add -

把下面的源添加到 /etc/apt/sources.list 文件中

deb http://dl.google.com/linux/deb/ stable non-free main

随后更新数据库

sudo apt-get update

安装软件

sudo apt-get install google-chrome-unstable
sudo apt-get install picasa
sudo apt-get install google-desktop-linux

Google earth 安装，试了一下，下面的方法可能会不成功，感觉比较麻烦，还是建议文件下载安装。

sudo apt-get install googleearth-package
sudo make-googleearth-package –force
sudo dpkg -i *.deb

参考：http://www.google.com/linuxrepositories/apt.html

来源：http://wiki.wowubuntu.com/howto/install_google_apps
