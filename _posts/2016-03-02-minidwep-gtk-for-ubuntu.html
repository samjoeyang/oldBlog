一、相关依赖
sudo apt-get install build-essential libssl-dev iw libpcap-dev sqlite3 libsqlite3-dev libpcap0.8-dev libssl0.9.8 libnl-dev

二、相关组件
sudo apt-get install aircrack-ng

sudo apt-get install reaver

下载地址https://sourceforge.net/projects/xiaopanos/files/Minidwep%2040420/
sudo dpkg -i minidwep-gtk-40420-ubuntu-64bit.deb

三、运行
sudo minidwep-gtk

四、问题
sudo minidwep-gtk无反应
minidwep-gtk的执行主脚本位于/usr/local/bin/minileafdwep/minidwep-gtk.sh
可执行这个脚本

原来Ubuntu没有安装gksu
sudo apt-get install gksu
sudo minidwep-gtk
正常运行
