Deditor 是一款专用于 Python 语言的轻量级编辑器，使用 Python 及 wxPython 开发。它主要功能有支持语法高亮、代码折叠、同时打开多文件、自动缩进等等。另外，最新的 0.2.3 版还提供了插件功能，插件的开发文档看这里。

项目主页： https://launchpad.net/deditor/

# 安装：

Deb 包下载： https://launchpad.net/deditor/+download

安装信赖包：

apt-get install wxgtk2.8
