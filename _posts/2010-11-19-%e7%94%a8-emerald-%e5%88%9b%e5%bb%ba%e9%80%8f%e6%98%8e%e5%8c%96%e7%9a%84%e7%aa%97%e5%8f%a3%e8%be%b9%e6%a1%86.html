Emerald 是一个窗口边框管理器，可以很好的与 Compiz 共同工作，通过它可以让窗口边框具备更加漂亮的透明等效果，以下是效果图。用户可以使用不同的 Theme 来获取不同的效果。

＃ 安装及设置：

1 安装 emerald

sudo apt-get install emerald

2 修改 Compiz 设置，让 emerald 来接管窗口边框，在 CompizConfig Settings Manager 中将 Window decorations 中的 Command 改为emerald – -replace（默认是：/usr/bin/compiz-decorator）即可自动开启emerald，然后重启。

3 到 Compiz-Theme 上下载自已喜欢的 emerald 边框主题，然后打开 Emerald theme manager 导入下载的主题并启用就要可以了。

4 另外，如果需要更好效果的话，可以安装 GlobalMenu

sudo add-apt-repository ppa:nilarimogard/webupd8

sudo apt-get update

sudo apt-get install gnome-globalmenu

安装完后重启，在顶部 Panel 上添加 Global Menu Panel Applet 就可以了。
