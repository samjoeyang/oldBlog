注:不包括ubuntu自带的指示器程序

1ubuntu one

功能:查看和显示你的ubuntu one账户同步情况

安装方法:here

2 触摸板指示器

功能:禁止和开启触摸板

安装方法:

sudo add-apt-repository ppa:lorenzo-carbonell/atareao
sudo apt-get update && sudo apt-get install touchpad-indicator
3 天气指示器
功能:显示有关天气的各个方面,功能强大
安装方法:
sudo add-apt-repository ppa:lorenzo-carbonell/atareao
sudo apt-get update && sudo apt-get install my-weather-indicator
4 cpu指示器
功能:可以调节主机的频率,方便节电

安装方法:

sudo add-apt-repository ppa:anton-sudak/indicators

sudo apt-get update && sudo apt-get install indicator-cpufreq

5 工作区指示器

功能:快捷转换工作区域

安装方法:here

6 键盘锁指示器

功能:显示caps,scroll,number这3个键的状态

安装方法:here

7 下面介绍一些剪贴板管理器

7.1 Pastie

功能:清除历史功能

安装方法:here

7.2 Diodon

功能:清量型的工具,和ubuntu默认主题配合不错

安装方法:here

7.3 Clip it

功能:从ParcelLite出来的一个分支,没有清除历史的选项,功能不够强大

下载:here

8 移除usb设备小程序

功能:显示挂载的usb设备,SD卡,卸载这些设备

下载:here

9 LookIt

功能:截图程序,简洁,可配置,方便,快速截图和上传图片

beta版安装方法:

sudo add-apt-repository ppa:lookit/testing
sudo apt-get update && sudo apt-get install lookit
stable版安装方法:
sudo add-apt-repository ppa:lookit/ppa
sudo apt-get update && sudo apt-get install lookit
10 VirtualBox指示器
功能:快速启动虚拟机,观察虚拟机的状态
11 壁纸切换小程序
功能:以DesktopNova为基础,使用简单,安装简便,乃换壁纸癖之人首选
安装方法:here
12 Caffeine
功能:让你的电脑不要休眠,可以防止在看视频或者电影时,系统的屏幕保护自动开启
安装方法:ppa:caffeine-developers/ppa添加到软件源进行安装
